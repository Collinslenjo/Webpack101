## Webpack boilerplate ##

>  Webpack101 is the basics of compiling using webpack2 
> - Using Loaders
> - Using Modules
> - And much more

### Getting started ###

 - ```git clone``` the app and ```cd``` into the app directory
 - run ```npm install``` and ```npm run build```

#### or use The Express server ####

 - run ``` npm start ```
